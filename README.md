# Ці малі одружуються

Весільний сайт-запрошення для пари. Зроблений як легка статична сторінка: без збірки, без серверу, без платних сервісів.

Майбутня GitHub Pages адреса:

```text
https://muddotaa.github.io/mali-odruzhuyutsya/
```

## Що всередині

- перший екран з намальованою парою;
- дата, час і місце весілля;
- пастельний дрес-код з інтерактивними палітрами;
- кнопка підтвердження присутності через Telegram.

## Файли

```text
index.html   основна сторінка
style.css    стилі
script.js    анімації та палітри дрес-коду
assets/      PNG-ілюстрації
```

## Локальний перегляд

Відкрий `index.html` у браузері.

## Публікація на GitHub Pages

1. Створи публічний репозиторій `mali-odruzhuyutsya` в акаунті `muddotaa`.
2. Завантаж ці файли в репозиторій.
3. Відкрий `Settings` -> `Pages`.
4. Обери `Deploy from a branch`.
5. Вкажи `main` і папку `/(root)`.
6. Натисни `Save`.

Після публікації сайт буде доступний за адресою:

```text
https://muddotaa.github.io/mali-odruzhuyutsya/
```

Оновлення на GitHub Pages можуть з'являтися не миттєво, іноді потрібно кілька хвилин.
